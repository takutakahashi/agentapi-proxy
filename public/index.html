🤖 AgentAPI Proxy - Mock API Server

注意: これはモックAPIサーバーです。実際のagentapi-proxyの動作をシミュレートしています。

📝 利用可能なエンドポイント

セッション管理

POST
/start
新しいセッションを作成します
POST
/start-with-profile
プロファイルを使用して新しいセッションを作成します
GET
/search
既存のセッション一覧を検索・取得します

プロファイル管理

POST
/profiles
新しいプロファイルを作成します
GET
/profiles
プロファイル一覧を取得します
GET
/profiles/:profileId
特定のプロファイルの詳細を取得します
PUT
/profiles/:profileId
既存のプロファイルを更新します
DELETE
/profiles/:profileId
プロファイルを削除します
POST
/profiles/:profileId/repositories
プロファイルにリポジトリエントリを追加します
POST
/profiles/:profileId/templates
プロファイルにメッセージテンプレートを追加します
