🤖 AgentAPI Proxy - Mock API Server (Netlify)

注意: これはNetlify上で動作するモックAPIサーバーです。実際のagentapi-proxyの動作をシミュレートし、フロントエンドのテストに使用できます。
使用方法: 以下のエンドポイントに直接HTTPリクエストを送信できます。例: POST /start, GET /profiles

📝 利用可能なエンドポイント

セッション管理

POST
/start
新しいセッションを作成します
POST
/start-with-profile
プロファイルを使用して新しいセッションを作成します
GET
/search
既存のセッション一覧を検索・取得します

プロファイル管理

POST
/profiles
新しいプロファイルを作成します
GET
/profiles
プロファイル一覧を取得します
GET
/profiles/:profileId
特定のプロファイルの詳細を取得します
PUT
/profiles/:profileId
既存のプロファイルを更新します
DELETE
/profiles/:profileId
プロファイルを削除します
POST
/profiles/:profileId/repositories
プロファイルにリポジトリエントリを追加します
POST
/profiles/:profileId/templates
プロファイルにメッセージテンプレートを追加します

🚀 使用例

セッション作成:

curl -X POST https://your-netlify-domain.netlify.app/start \
  -H "Content-Type: application/json" \
  -d '{"user_id": "test-user", "tags": {"env": "test"}}'

プロファイル一覧取得:

curl https://your-netlify-domain.netlify.app/profiles
